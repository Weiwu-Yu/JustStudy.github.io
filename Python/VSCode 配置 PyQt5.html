如何像配置pycharm一样在vscode中配置pyqt5的designer
ChatGPT said:

明白了！你想要在 VSCode 中，像 PyCharm 那样，舒服地使用 Qt Designer 来设计 .ui 文件，并且让它们跟你的 PyQt5 项目流畅地配合起来，对吧？

可以，步骤不多，我帮你一条一条理一下：

第一步：安装 VSCode 必备插件

  1. Python Extension（微软官方出的，支持 Python 语法高亮、运行等）

  2. Qt for Python（插件名叫："Qt for Python"，作者是 seanwu）
    👉 支持 .ui 文件预览和编辑

  3. (可选) Code Runner（可以快速运行 Python 文件）

第二步：安装 Qt Designer 工具

PyCharm 自带，但 VSCode 需要自己手动配一下：

  • 方法 1：如果你有安装 Anaconda，直接用里面的 Designer。

    • 在 Anaconda Prompt 里输入：

      bash
      designer

  • 方法 2：单独下载：

    • 下载 Qt（建议下 5.15 LTS版）：https://download.qt.io/archive/qt/

    • 装好后，找到 designer.exe，路径一般在：

      makefile
      C:\Qt\5.15.2\mingw81_64\bin\designer.exe

第三步：VSCode里关联 .ui 文件

👉 方法 1：直接右键 .ui 文件 → 配置成外部打开

  • 打开你的 VSCode settings.json （Ctrl+Shift+P → 搜索 Settings → 选 Open Settings (JSON)）

  • 加一段配置，让 .ui 文件用外部 Designer 打开：

json
"files.associations": {
    "*.ui": "xml"
},
"qt.toolsPath": "C:/Qt/5.15.2/mingw81_64/bin", 
"qt.designerPath": "C:/Qt/5.15.2/mingw81_64/bin/designer.exe"

  • 注意：路径换成你自己实际安装的位置。

这样 .ui 文件就能在 Designer 中正常打开了！

第四步：把 .ui 文件转成 .py 文件（自动化）

Designer 只是做界面，PyQt5 需要把 .ui 转 .py 才能用。

在 VSCode 里，可以：

  1. 打开终端 (Ctrl+`)，输入：

    bash
    pyuic5 -o output.py input.ui

    例如：

    bash
    pyuic5 -o mywindow.py mywindow.ui

  2. 想更省事？可以配置任务（自动化一键转）：

  • 打开 VSCode → Terminal → Configure Default Build Task → Create tasks.json

  • 配置如下：

json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "Convert UI to PY",
            "type": "shell",
            "command": "pyuic5",
            "args": [
                "-o",
                "${fileDirname}/${fileBasenameNoExtension}.py",
                "${file}"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "presentation": {
                "echo": true,
                "reveal": "always",
                "focus": false,
                "panel": "shared"
            },
            "problemMatcher": []
        }
    ]
}

  • 用法：打开 .ui 文件，按 Ctrl+Shift+B，一键转成 .py！

1. 安装 PyQt5 / PySide2 / PySide6

打开 VSCode 的终端（Terminal）执行：

bash
pip install PyQt5 pyqt5-tools

（或者如果你用的是 PySide，可以用 pip install PySide2 或 pip install PySide6）

✅ 这步主要是安装 Python 的 Qt 绑定库。

2. 安装 Qt for Python 插件（vscode扩展）

在 VSCode 里：

  • 打开 Extensions （扩展）

  • 搜索 Qt for Python

  • 安装！

✅ 这步让 VSCode 支持 .ui 文件、支持自动转 .py、支持一键打开 Designer。

3. 配置 Qt Designer 路径

打开 VSCode 设置（Ctrl + ,）
搜索 qtForPython

在 Designer: Path 这一栏，填上 Qt Designer 的实际路径，比如：

bash
C:/Qt/5.15.2/mingw81_64/bin/designer.exe

✅ 这步让 VSCode 知道怎么打开 Qt Designer。

如果没有装Qt5完整套件，也可以用 pyqt5-tools 自带的 Designer，比如路径是：

bash
C:/Users/你的用户名/AppData/Local/Programs/Python/Python版本/Lib/site-packages/pyqt5_tools/designer.exe

小技巧：如果找不到 designer.exe，我可以教你快速找出来！

4. 打开 .ui 文件 ➔ 画界面！

现在，在 VSCode 中：

  • 新建一个 .ui 文件（或者用现成的）

  • 右键 → 找到 “Open in Qt Designer” （用插件打开）

  • 开始拖拽控件、保存

✅ 这步就可以像 PyCharm 那样愉快地画界面了！

5. 自动生成 Python 代码

保存 .ui 文件后，VSCode插件会自动帮你生成同名的 _ui.py 文件，比如：

bash
mainwindow.ui  ➔  mainwindow_ui.py

如果没自动生成，也可以手动在终端执行：

bash
pyuic5 mainwindow.ui -o mainwindow_ui.py

6. 写你的主程序，导入UI

例如：

python
from PyQt5.QtWidgets import QApplication, QMainWindow
from mainwindow_ui import Ui_MainWindow

app = QApplication([])
window = QMainWindow()
ui = Ui_MainWindow()
ui.setupUi(window)
window.show()
app.exec_()

然后在 VSCode 里直接运行！
